# AIGC Showcase UI Design Preview

归档时间：2026-06-05

本目录归档 MVP 迭代升级阶段的第一组 UI 设计图。设计方向为“影像厂牌 / AIGC 创作档案馆”，保留黑底与荧光黄 `#E8FF00` 的品牌识别，重点强化首页视频张力、作品展示质感和专业制作能力表达。

## Files

- `home-desktop.png`：首页 Desktop 设计图，包含首屏视频 Hero、导航、作品筛选、作品卡片和空状态。
- `work-detail-desktop.png`：作品详情页 Desktop 设计图，包含主视频、项目信息、工具/主创信息、剧照、海报和创作手记入口。
- `mobile-home-and-detail.png`：移动端首页与作品详情双屏设计图。
- `about-and-design-system.png`：About 页面与基础设计系统，包含字标、色板、字体方向和组件状态。

## Frontend Notes

- 首页首屏应使用全屏视频或视频首帧作为主视觉，并叠加上下暗部渐变，保证不同明暗素材下文字可读。
- 作品卡片保留影像主导，Hover 或 Active 状态可使用荧光黄描边、播放图标和轻微遮罩。
- 详情页优先呈现视频播放器，项目信息以影片档案方式组织，不做营销式大段卖点。
- About 页面以团队宣言、能力分区、工具体系和商业合作入口为主。
- 圆角建议控制在 `4px` 左右，边框细、留白克制，避免卡片套卡片。
